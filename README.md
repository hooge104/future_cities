# Interactive explorer of future climates of the World's major cities

Hosted on shinyapps.io: https://hooge104.shinyapps.io/css-app/

To run in Rstudio, run:

```
runGitHub("future_cities", "hooge104")
```

Required packages:

```
library(shiny)
library(leaflet)
library(tidyverse)
library(RColorBrewer)
library(htmltools)
library(DT)
library(reshape2)
```
